Features:
- Authentication flow
- Protected routes
- Customers & Orders management
- Detail pages
- Theme switching (light/dark)
- Responsive layout
- Status badges, tables, charts
- Realistic UX patterns

Tech stack:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Zustand
- next-themes

This project simulates a real internal tool used to manage customers and orders.
